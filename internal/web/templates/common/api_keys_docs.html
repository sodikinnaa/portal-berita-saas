Autentikasi API

Semua request API wajib menyertakan API Key aktif yang dikirim melalui HTTP Header Authorization menggunakan format Bearer Token.

Header Contoh
Authorization: Bearer portal_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Pastikan API Key Anda memiliki scope yang sesuai dengan endpoint yang akan dipanggil.

Buat Artikel Baru

Scope wajib: articles:create

POST /api/v1/articles

Membuat artikel baru ke database. Secara default, artikel yang dibuat via API akan berstatus draft dengan penulis (author) diset ke admin pemilik API Key.

Payload Parameters

Field Type Wajib Keterangan
title string Ya Judul utama artikel.
slug string Tidak Slug unik artikel. Jika kosong, slug otomatis dibuat dari judul.
excerpt string Tidak Ringkasan artikel yang muncul di halaman depan.
content string Ya Konten/isi artikel lengkap. Mendukung teks biasa atau HTML aman.
category string Ya Kategori artikel (contoh: Teknologi, Bisnis).
hero_image_url string Tidak URL untuk gambar utama artikel. Bisa memakai hasil upload media.
status string Tidak Status artikel (draft, submitted, atau published). Default: draft.

Contoh Request

curl -X POST http://localhost:8080/api/v1/articles \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Tren Kecerdasan Buatan di Tahun 2026",
    "slug": "tren-ai-2026",
    "excerpt": "Perkembangan AI semakin mengubah produktivitas kerja sehari-hari.",
    "content": "Kecerdasan buatan mengalami lompatan besar terutama dalam agen otonom...",
    "category": "Teknologi",
    "hero_image_url": "",
    "status": "draft"
  }'
package main

import (
	"bytes"
	"encoding/json"
	"fmt"
	"net/http"
)

func main() {
	url := "http://localhost:8080/api/v1/articles"
	payload := map[string]string{
		"title":    "Tren Kecerdasan Buatan di Tahun 2026",
		"content":  "Kecerdasan buatan mengalami lompatan besar terutama dalam agen otonom...",
		"category": "Teknologi",
		"status":   "draft",
	}
	body, _ := json.Marshal(payload)

	req, _ := http.NewRequest("POST", url, bytes.NewBuffer(body))
	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Content-Type", "application/json")

	resp, err := (&http.Client{}).Do(req)
	if err != nil {
		fmt.Println("Error:", err)
		return
	}
	defer resp.Body.Close()
	fmt.Println("Response Status:", resp.Status)
}
const axios = require('axios');

axios.post('http://localhost:8080/api/v1/articles', {
  title: 'Tren Kecerdasan Buatan di Tahun 2026',
  content: 'Kecerdasan buatan mengalami lompatan besar terutama dalam agen otonom...',
  category: 'Teknologi',
  status: 'draft'
}, {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
})
.then(res => console.log('Sukses:', res.data))
.catch(err => console.error('Gagal:', err.response?.data || err.message));

Upload Media Gambar

Scope wajib: media:upload

POST /api/v1/media/upload

Mengunggah file media gambar lokal ke server portal berita. Format request wajib berupa multipart/form-data.

Payload Parameters

Field Type Wajib Keterangan
file file (binary) Ya File gambar yang akan diunggah (JPG, PNG, JPEG, maks 5MB).

Contoh Request

curl -X POST http://localhost:8080/api/v1/media/upload \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@/path/ke/gambar-anda.jpg"
package main

import (
	"bytes"
	"fmt"
	"io"
	"mime/multipart"
	"net/http"
	"os"
)

func main() {
	file, _ := os.Open("gambar.jpg")
	defer file.Close()

	body := &bytes.Buffer{}
	writer := multipart.NewWriter(body)
	part, _ := writer.CreateFormFile("file", "gambar.jpg")
	io.Copy(part, file)
	writer.Close()

	req, _ := http.NewRequest("POST", "http://localhost:8080/api/v1/media/upload", body)
	req.Header.Set("Authorization", "Bearer YOUR_API_KEY")
	req.Header.Set("Content-Type", writer.FormDataContentType())

	resp, err := (&http.Client{}).Do(req)
	if err != nil {
		fmt.Println("Error:", err)
		return
	}
	defer resp.Body.Close()
	fmt.Println("Response Status:", resp.Status)
}
const axios = require('axios');
const FormData = require('form-data');
const fs = require('fs');

const form = new FormData();
form.append('file', fs.createReadStream('gambar.jpg'));

axios.post('http://localhost:8080/api/v1/media/upload', form, {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    ...form.getHeaders()
  }
})
.then(res => console.log('Sukses:', res.data))
.catch(err => console.error('Gagal:', err.response?.data || err.message));

Daftarkan URL Media Eksternal

Scope wajib: media:url

POST /api/v1/media/url

Mendaftarkan metadata gambar yang di-hosting di server eksternal, tanpa perlu mengunggah file fisiknya ke server portal berita.

Payload Parameters

Field Type Wajib Keterangan
url string Ya URL lengkap lokasi file gambar berada (harus diawali http:// atau https://).
original_name string Tidak Nama berkas asli untuk disimpan dalam metadata.

Contoh Request

curl -X POST http://localhost:8080/api/v1/media/url \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://images.unsplash.com/photo-1504711434969-e33886168f5c",
    "original_name": "berita-utama.jpg"
  }'
