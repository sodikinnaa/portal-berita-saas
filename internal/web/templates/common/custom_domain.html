Pengaturan Domain

Hubungkan Domain Baru

Masukkan domain utama tanpa protokol (tanpa http:// atau https://). Contoh: beritaku.com atau news.beritaku.com.

Konfigurasi DNS

Panduan Pointing DNS

Sebelum menghubungkan domain, Anda wajib menambahkan DNS Record berikut di panel registrar tempat Anda membeli domain (seperti Niagahoster, Namecheap, dll):

Tipe Nama/Host Value/Target
A @ (atau kosong) {{.IPAddress}}
CNAME www @ (atau domain Anda)

🔒 Masa Aktif & Perpanjangan SSL

Sertifikat SSL Let's Encrypt / ZeroSSL aktif selama 90 hari. Namun, sistem (Caddy) akan otomatis memperbarui (renew) sertifikat tersebut 30 hari sebelum kedaluwarsa secara berkala di latar belakang tanpa downtime.

⚠️ Bagaimana Jika SSL Mati/Error?

Jika SSL terdeteksi tidak aktif atau menampilkan peringatan tidak aman, periksa hal berikut:

  1. DNS Record Terputus: Pastikan Anda tidak menghapus A Record di panel DNS domain Anda. DNS wajib selalu mengarah ke IP {{.IPAddress}} agar perpanjangan otomatis berhasil.
  2. Fitur Proxy Cloudflare Aktif: Jika menggunakan Cloudflare, pastikan status proxy diubah ke "DNS Only" (awan abu-abu) atau atur enkripsi SSL/TLS Cloudflare ke mode "Full (Strict)".
  3. Masa Aktif Domain Habis: Pastikan domain Anda tidak kedaluwarsa di registrar tempat Anda membelinya.
