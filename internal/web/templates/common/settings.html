Informasi Umum

Pengaturan identitas menentukan nama portal berita, favicon di tab peramban, slogan situs, serta deskripsi di bagian kaki halaman (footer).

  • Judul Utama: Digunakan di logo teks, nama brand, dan meta title SEO.

  • Favicon: Icon kecil situs di samping judul tab browser.

Kecerdasan Buatan

Kredensial OpenAI-Compatible / Gemini API

Masukkan API Key dari penyedia layanan AI. Biarkan kosong jika ingin menggunakan mode Mock AI.
Jika menggunakan Gemini API langsung, isi: https://generativelanguage.googleapis.com/v1beta/models/. Jika OpenAI-compatible, isi base URL endpoint-nya.
Pilih model default untuk asisten tulis. Klik "Tarik Model" untuk memvalidasi kunci dan memuat daftar model yang tersedia.
Gunakan proxy acak yang aktif dari Proxy Rotator untuk semua panggilan API asisten AI (seperti rewrite artikel, generate caption Facebook, dll.). Berguna jika IP server Anda terblokir atau dibatasi kuota oleh penyedia AI.

Panduan Integrasi AI

Asisten AI digunakan untuk menyusun ulang artikel (rewrite), menulis caption berita ke media sosial dalam bahasa Inggris, serta memoles ringkasan secara instan.

  • Gemini API: Layanan bawaan dan terekomendasi untuk auto-post sosial.
  • OpenRouter: Memberikan akses ke ratusan model alternatif (seperti Claude, Llama).
  • Mock AI: Mode simulasi lokal gratis jika Anda belum memiliki API Key.

Portal berita ini secara otomatis menghasilkan file /sitemap.xml yang dinamis untuk melacak seluruh tulisan aktif Anda.

Dengan menempelkan kode verifikasi di samping, robot perayap Google dan Bing akan mengenali kepemilikan situs Anda secara instan dan mempercepat indeksasi berita.

Backup & Restore Database

Cadangkan seluruh data portal ke file eksternal atau pulihkan data dari file JSON cadangan Anda.

Ekspor Database (.json)

Unduh file cadangan data lengkap (artikel, kategori, media, api_keys, blacklist, dan konfigurasi) dalam skema JSON yang aman.

Unduh Backup ⬇️

Impor/Restore Database

Unggah file JSON hasil ekspor sebelumnya untuk memulihkan database. ⚠️ PERINGATAN: Tindakan ini akan menghapus seluruh data yang ada saat ini secara permanen! Pastikan file backup valid.

Tips Keamanan

Selalu ekspor database Anda secara berkala sebelum melakukan upgrade binary, modifikasi source code, atau migrasi server guna menghindari kehilangan data yang tidak terduga.
