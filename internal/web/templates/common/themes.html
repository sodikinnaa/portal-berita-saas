Unggah Custom Homepage (.zip)

Unggah file .zip yang berisi file index.html pada root zip (wajib), beserta file pendukung seperti style.css, script.js, dan folder assets/.

💡 Fitur Dynamic Templating Aktif

Custom Homepage Anda kini **mendukung Go HTML Templating** secara dinamis.

Anda dapat menyisipkan kode Go Template seperti {{"{{"}}range .LatestArticles{{"}}"}} untuk menampilkan artikel terbaru dari database, dan {{"{{"}}range .Categories{{"}}"}} untuk menampilkan menu navigasi kategori.
